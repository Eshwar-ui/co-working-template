## ✨ Key Features
- **Modern Design**: Clean, minimal, and premium aesthetic.
- **Responsive**: Fully optimized for all device sizes.
- **Dark Mode**: Native dark mode support with theme persistence.
- **RTL Support**: Built-in Right-to-Left layout toggle.
- **Dashboards**: Comprehensive User and Admin dashboards.
- **SEO Optimized**: Optimized meta tags, sitemap, and JSON-LD structured data.
- **Animations**: Smooth GSAP scroll animations.
- **Organized Structure**: Logically categorized pages into subdirectories for better maintainability.

## 📂 Project Structure
```text
├── assets/             # Static assets (css, js, images)
├── pages/              # Reorganized page modules
│   ├── auth/           # Login, Signup
│   ├── blog/           # Articles and details
│   ├── dashboard/      # Admin & User panels
│   ├── events/         # Community meetups
│   ├── spaces/         # Workspace listings
│   └── utility/        # 404, Gallery, etc.
├── documentation/      # Setup and customization guides
├── index.html          # Root redirect
└── sitemap.xml         # SEO sitemap
```
