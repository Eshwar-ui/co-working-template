# Installation Guide

Follow these steps to set up the CoWork Hub template on your local machine or server.

## 📋 Prerequisites
No special server-side software is required. You only need:
- A modern web browser (Chrome, Firefox, Safari, Edge).
- A code editor (VS Code, Sublime Text, etc.) if you wish to modify the code.

## 🚀 Step-by-Step Setup

1. **Extract the Files**:
   Unzip the downloaded template package into your desired project directory.

2. **Open in Browser**:
   - Locate the `index.html` file in the root directory.
   - Right-click and "Open with" your preferred web browser.

3. **Local Development (Optional but Recommended)**:
   For the best experience (especially for animations and external assets), use a local server:
   - If using VS Code, install the **Live Server** extension.
   - Open the project folder in VS Code, right-click `index.html`, and select "Open with Live Server".

4. **Deployment**:
   To make your site live:
   - Upload all files from the project directory to your web server (via FTP, GitHub Pages, Netlify, or Vercel).
   - Ensure the `assets/` folder is uploaded as it contains all styles, scripts, and images.
