# Support

Need help with the CoWork Hub template?

## 🛠️ Common Troubleshooting
- **Animations not working?** Ensure you have an active internet connection to load GSAP from the CDN.
- **RTL looks strange?** Clear your browser cache and local storage (`localStorage.clear()`) to reset the toggle state.
- **Tailwind classes not applying?** Check the console for any script loading errors.
