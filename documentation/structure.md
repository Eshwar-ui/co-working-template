# Page Structure

Explanation of the files and directories in the CoWork Hub template.

## 📂 Directory Hierarchy
- `/` - Root directory containing core index and meta files.
- `/assets/` - All static assets (css, js, images, fonts).
- `/pages/` - Reorganized internal pages.
    - `/auth/` - Login and Signup pages.
    - `/blog/` - Blog listing and details.
    - `/dashboard/` - Admin and User dashboards.
    - `/events/` - Community events and details.
    - `/spaces/` - Workspace listings and amenities.
    - `/utility/` - 404, Gallery, Coming Soon, etc.
- `/documentation/` - Project documentation.

## 📄 Key Pages

### Core Pages
- `index.html`: (Root) Redirects to `/pages/index.html`.
- `pages/index.html`: Primary landing page.
- `pages/about.html`: Company mission and values.
- `pages/contact.html`: Location and inquiry form.

### Feature Sections
- `pages/spaces/index.html`: Available workspace listings.
- `pages/blog/index.html`: Community insights and news.
- `pages/events/index.html`: Upcoming community gatherings.

### Dashboards
- `pages/dashboard/user/index.html`: Member personal dashboard.
- `pages/dashboard/admin/index.html`: Administrative oversight.
